About the world
A tiny voxel sandbox running in a single file
Client‑side · WebGL

VOXEL·31 is a compact, Minecraft‑style engine that lives entirely inside an HTML document. It boots up a full 3D world, handles terrain, blocks, inventory, crafting, and furnaces, and renders everything via WebGL in your browser. [file:1]

Because it is just HTML + JavaScript, you can host it on any static site service, drop it into your own portfolio, or even ship it on a USB stick as a portable playground. [file:1]
